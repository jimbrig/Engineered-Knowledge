# Robocopy

> `robocopy` is amazing.

## Examples

```powershell
robocopy "C:\bin" "C:\Users\jimmy\OneDrive\Backups\bin" /E
```

*Note: the `/E` flag is used to recursively copy directory contents from one directory to another (including file data, attributes and timestamps).*

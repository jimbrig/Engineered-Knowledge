# Windows User Dictionaries Locations

- Windows: 
	- `$env:APPDATA\Microsoft\Spelling\en-US\default.dic`
- Typora:
	- `$env:APPDATA\Typora\typora-dictionaries\user-dict.json`
	- `$env:APPDATA\Typora\typora-dictionaries\en_US.dic`

	-
